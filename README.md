# 🔍 Deepfake & AI-Generated Media Detection (Kaggle Experiments)

This repository contains a collection of **Kaggle notebooks** exploring different approaches to detect **deepfakes and AI-generated images/videos** using both **pretrained models and custom deep learning architectures**.

---

## 📁 Contents

### 1. Video Deepfake Detection (EfficientNetB0)
- Uses **EfficientNetB0 pretrained model**
- Frame-based video classification
- Detects manipulated video content (deepfake vs real)

---

### 2. Deepfake Detection with CNN (From Scratch)
- Custom **Convolutional Neural Network (CNN)**
- Trained on image datasets
- Focus on understanding deepfake classification from fundamentals

---

### 3. Real vs AI-Generated Image Classification
- Binary classification task:
  - Real images vs AI-generated images
- Uses preprocessing + augmentation techniques
- Experimentation with different model architectures

---

## 🛠️ Technologies Used

- Python

- OpenCV
- NumPy / Pandas
- Matplotlib
- Jupyter Notebook (Kaggle environment)
